PEC2 Visualización de Datos

Andreu Molina Espi

Descripción

En esta PEC se van a representar las siguientes técnicas de visualización de datos:

  • Line Chart
  • Arc Diagram
  • Beeswarm Plot

Herramientas utilizadas

Para la realización de estos gráficos se ha hecho uso de la plataforma RAWGraphs (https://app.rawgraphs.io/) escogiendo datos de muestra para cada uno de los gráficos

  • Line Chart: Revenues per Music format.

    La fuente de estos datos proviene de Recording Industry Association of America (RIIA). Puedes consultar la base de datos en el siguiente enlace: RIAA Sales Database.

  • Arc Diagram: Lannister vs Stark Relationship.

    La fuente de estos datos proviene de un dataset subido a Kaggle: Ver dataset.

  • Beeswarm Plot: New York mean wage per occupation.

    La fuente de estos datos proviene de Occupational Employment and Wage Statistics: Consultar datos.
